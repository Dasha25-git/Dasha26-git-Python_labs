# Лабораторные работы по Python ООП

Предметная область: **библиотека / книги**.

## Структура проекта

```text
python_labs/
├─ README.md
├─ src/
│  ├─ lab01/
│  ├─ lab02/
│  ├─ lab03/
│  ├─ lab04/
│  ├─ lab05/
│  └─ lab06/
└─ images/
   ├─ lab01/
   ├─ lab03/
   ├─ lab04/
   ├─ lab05/
   └─ lab06/
```

## Содержание

* ЛР-1 — класс `Book`, инкапсуляция, свойства, магические методы и бизнес-логика.
* ЛР-2 — контейнер `Library` для хранения объектов `Book`.
* ЛР-3 — наследование: `PrintedBook`, `EBook`, `AudioBook`.
* ЛР-4 — интерфейсы и абстрактные классы через `ABC`.
* ЛР-5 — функции как аргументы, `map`, `filter`, `lambda`, стратегии.
* ЛР-6 — аннотации типов, `Generic`, `TypeVar`, `Protocol`.

## Запуск

Из папки `python_labs`:

python3 -m src.lab01.demo
python3 -m src.lab02.demo
python3 -m src.lab03.demo
python3 -m src.lab04.demo
python3 -m src.lab05.demo
python3 -m src.lab06.demo
