# Лабораторные работы по Python ООП

Готовый репозиторий с лабораторными работами по предметной области **библиотека / книги**.

## Структура

```text
python_labs/
├─ README.md
├─ README_LABS_3_5.md
├─ src/
│  ├─ lab01/
│  ├─ lab02/
│  ├─ lab03/
│  ├─ lab04/
│  ├─ lab05/
│  └─ lab06/
└─ images/
   ├─ lab01/
   ├─ lab03/
   ├─ lab04/
   ├─ lab05/
   └─ lab06/
```

## Запуск

Из папки `python_labs`:

```bash
python3 -m src.lab01.demo
python3 -m src.lab02.demo
python3 -m src.lab03.demo
python3 -m src.lab04.demo
python3 -m src.lab05.demo
python3 -m src.lab06.demo
```

## Содержание

* ЛР-1 — класс `Book` и инкапсуляция.
* ЛР-2 — коллекция `Library`.
* ЛР-3 — наследование: `PrintedBook`, `EBook`, `AudioBook`.
* ЛР-4 — интерфейсы через `ABC`.
* ЛР-5 — функции как аргументы и паттерн «Стратегия».
* ЛР-6 — `typing`, `Generic`, `TypeVar` и `Protocol`.
